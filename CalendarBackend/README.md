# CalendarBackend
Base URL: http://api.localhost:3000/v1

| Method | Endpoint       | Usage           |
|:-------|:---------------|:----------------|
| GET    | /v1/events     | Get all events  |
| POST   | /v1/events     | Create an event |
| PUT    | /v1/events/:id | Edit an event   |
| DELETE | /v1/events/:id | Delete an event |
